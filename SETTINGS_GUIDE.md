# Settings & Module Management Guide

## Overview
The Settings page provides comprehensive control over your MuktiAp ERP system, allowing you to enable/disable modules, manage users, configure integrations, and more.

## Accessing Settings
1. Log into your dashboard
2. Click the **Settings** icon in the sidebar (gear icon)
3. Or use the keyboard shortcut: `Ctrl/Cmd + ,`

## How to Enable/Disable Modules

1. Navigate to **Settings** â†’ **Modules** tab
2. Find the module you want to enable/disable
3. Click the toggle switch on the right side
4. The module will be immediately enabled/disabled
5. Click **Save Changes** to persist your settings

## Module Status Indicators

- **Enabled** (Blue toggle) - Module is active and accessible
- **Disabled** (Gray toggle) - Module is inactive and hidden from navigation

## Best Practices

1. **Start Small** - Enable only the modules you need initially
2. **Review Regularly** - Periodically review enabled modules to optimize performance
3. **Train Users** - Ensure team members are trained on enabled modules
4. **Test Changes** - Test module changes in a non-production environment first

## Troubleshooting

### Module Not Appearing After Enabling
- Refresh the page (F5)
- Clear browser cache
- Log out and log back in

### Cannot Disable a Module
- Some core modules cannot be disabled
- Check if other modules depend on it
- Contact support if issues persist
